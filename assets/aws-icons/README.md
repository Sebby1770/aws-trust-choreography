# AWS Architecture Icons

This directory contains the SVG assets used by AWS Flow Studio.

- Source: [AWS Architecture Icons](https://aws.amazon.com/architecture/icons/)
- Package: 2026 Q2, published April 30, 2026
- Imported assets: 305 service icons, 516 resource icons, 15 group icons, and 26 category icons
- Catalog: `catalog.js`, generated by `node tools/build-aws-icon-catalog.mjs`

AWS provides these approved assets for architecture diagrams. Keep the source package and release metadata current when refreshing the catalog.
